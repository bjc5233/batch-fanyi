# batch-fanyi


## 说明
batch下英文单词翻译工具。调用的是有道的接口
* 安装JRE环境
* fanyi.bat english-word
* 建议生成fanyi.bat快捷方式为fy.lnk，存储到系统环境变量中，这样就可以在[运行]窗口中调用[fy love]
